# دليل إصلاح DarbStu — مرجع شامل
> يُستخدم هذا الدليل عند زيارة جهاز السيرفر أو أي جهاز آخر.
> نفّذ الخطوات بالترتيب وتوقف عند أول نجاح.

---

## أولاً: التشخيص السريع (ابدأ هنا دائماً)

افتح **Command Prompt** (cmd) كـ Administrator وشغّل هذه الأوامر:

```cmd
:: هل uvicorn يعمل على المنفذ 8000؟
curl -s -o nul -w "%{http_code}" http://localhost:8000/
:: النتيجة المتوقعة: 200 أو 302 — أي شيء آخر = uvicorn متوقف

:: هل DarbStu.exe يعمل؟
tasklist | findstr DarbStu

:: هل cloudflared يعمل؟
tasklist | findstr cloudflared

:: ما المنافذ المستخدمة؟
netstat -ano | findstr :8000
```

---

## الحل 1: إعادة تشغيل DarbStu.exe (الأبسط)

```cmd
taskkill /F /IM DarbStu.exe
timeout /t 3
start "" "C:\Users\%USERNAME%\Desktop\DarbStu\DarbStu.exe"
```

انتظر 30 ثانية ثم اختبر: افتح المتصفح على `http://localhost:8000`

---

## الحل 2: فحص uvicorn بشكل مستقل

إذا كان DarbStu.exe يعمل لكن localhost:8000 لا يستجيب:

```cmd
:: شغّل uvicorn يدوياً لرؤية الأخطاء
cd "C:\Users\%USERNAME%\Desktop\DarbStu"
python -m uvicorn server:app --host 127.0.0.1 --port 8000
```

راقب الأخطاء في النافذة — إذا ظهر خطأ مثل:
- `Address already in use` → انتقل للحل 3
- `ModuleNotFoundError` → انتقل للحل 7
- `No such file` → تأكد من المسار

---

## الحل 3: تحرير المنفذ 8000 المحجوز

```cmd
:: أوجد العملية التي تستخدم المنفذ 8000
netstat -ano | findstr :8000

:: ستظهر نتيجة مثل: TCP 0.0.0.0:8000 ... 1234
:: الرقم الأخير (1234) هو PID — أوقفه:
taskkill /F /PID 1234

:: ثم أعد تشغيل DarbStu
start "" "C:\Users\%USERNAME%\Desktop\DarbStu\DarbStu.exe"
```

---

## الحل 4: إعادة تشغيل cloudflared فقط

إذا كان localhost:8000 يعمل لكن الموقع الخارجي لا:

```cmd
taskkill /F /IM cloudflared.exe
timeout /t 3

:: إذا كان cloudflared مضبوط كـ Named Tunnel (config.yml):
cloudflared tunnel run darbte

:: إذا أردت Quick Tunnel مؤقتاً:
cloudflared tunnel --url http://localhost:8000
```

انتظر 30 ثانية ثم اختبر darbte.uk

---

## الحل 5: إعادة تشغيل كامل (DarbStu + cloudflared)

```cmd
taskkill /F /IM DarbStu.exe
taskkill /F /IM cloudflared.exe
taskkill /F /IM python.exe
timeout /t 5
start "" "C:\Users\%USERNAME%\Desktop\DarbStu\DarbStu.exe"
```

انتظر 40 ثانية كاملة ثم اختبر.

---

## الحل 6: مسح إعدادات cloudflared الفاسدة

```cmd
taskkill /F /IM cloudflared.exe
timeout /t 2

:: احذف مجلد الإعدادات القديمة
rmdir /S /Q "%USERPROFILE%\.cloudflared"

:: إذا كان يوجد config.yml في مجلد DarbStu احتفظ به
:: أعد تشغيل DarbStu
start "" "C:\Users\%USERNAME%\Desktop\DarbStu\DarbStu.exe"
```

---

## الحل 7: تنزيل cloudflared جديد

```cmd
taskkill /F /IM cloudflared.exe

:: احذف الملف القديم إن وُجد
del /F "C:\Users\%USERNAME%\Desktop\DarbStu\cloudflared.exe"

:: نزّل نسخة جديدة
curl -L --insecure --max-time 120 ^
  -o "C:\Users\%USERNAME%\Desktop\DarbStu\cloudflared.exe" ^
  "https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-windows-amd64.exe"

:: اختبر:
"C:\Users\%USERNAME%\Desktop\DarbStu\cloudflared.exe" --version

:: أعد تشغيل DarbStu
start "" "C:\Users\%USERNAME%\Desktop\DarbStu\DarbStu.exe"
```

---

## الحل 8: فحص Windows Firewall

```cmd
:: افتح Firewall Settings
firewall.cpl

:: أو تحقق هل المنفذ 8000 مسموح به:
netsh advfirewall firewall show rule name="DarbStu"

:: أضف استثناء إذا لم يوجد:
netsh advfirewall firewall add rule name="DarbStu" ^
  dir=in action=allow protocol=TCP localport=8000
```

## الحل 9: تشخيص مشكلة Python

إذا كان DarbStu.exe لا يبدأ أصلاً:

```cmd
cd "C:\Users\%USERNAME%\Desktop\DarbStu"

:: تحقق هل Python موجود:
python --version

:: شغّل main.py مباشرة لرؤية الأخطاء:
python main.py
```

## الحل 10: إعادة تثبيت كاملة

```cmd
:: 1. احتفظ بمجلد data (يحتوي بيانات الطلاب)
xcopy "C:\Users\%USERNAME%\Desktop\DarbStu\data" ^
      "C:\Users\%USERNAME%\Desktop\DarbStu_backup_data\" /E /I

:: 2. احذف مجلد DarbStu القديم
rmdir /S /Q "C:\Users\%USERNAME%\Desktop\DarbStu"

:: 3. نزّل النسخة الجديدة من GitHub وافتح المُثبّت

:: 4. بعد التثبيت، انسخ البيانات القديمة:
xcopy "C:\Users\%USERNAME%\Desktop\DarbStu_backup_data\" ^
      "C:\Users\%USERNAME%\Desktop\DarbStu\data\" /E /I
```

## مرجع سريع — جدول المشاكل والحلول

| المشكلة | الحل |
|---------|------|
| darbte.uk يعطي 502 Bad Gateway | ابدأ بالتشخيص السريع |
| localhost:8000 لا يستجيب | الحل 2 أو 3 |
| DarbStu.exe لا يظهر في tasklist | الحل 1 |
| cloudflared.exe لا يظهر في tasklist | الحل 4 |
| cloudflared يعمل لكن النفق منقطع | الحل 6 ثم 7 |
| المنفذ 8000 محجوز بعملية أخرى | الحل 3 |
| DarbStu.exe يفتح ويغلق فوراً | الحل 9 |
| أي شيء آخر لم يُحل | الحل 10 (إعادة تثبيت) |
