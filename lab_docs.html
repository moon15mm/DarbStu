⚙️ إعداد Google Drive

لحفظ البيانات على Google Drive تحتاج Client ID مجاني من Google Cloud Console.

  1. افتح: console.cloud.google.com
  2. أنشئ مشروعاً جديداً (New Project)
  3. من القائمة: APIs & Services → OAuth consent screen → اختر External → احفظ
  4. من القائمة: APIs & Services → Credentials → Create Credentials → OAuth Client ID
  5. اختر Web application
  6. في Authorized JavaScript origins أضف رابط موقعك (مثلاً https://yoursite.netlify.app)
